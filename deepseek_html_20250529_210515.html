梯形纹理坐标计算

本演示展示了如何计算梯形内任意点的纹理坐标。使用双线性插值方法，通过求解参数(s,t)将梯形映射到单位正方形，然后根据四个顶点的纹理坐标进行插值。

双线性插值公式

P = (1-s)(1-t)*A + s(1-t)*B + s*t*C + (1-s)t*D
UV = (1-s)(1-t)*UV_A + s(1-t)*UV_B + s*t*UV_C + (1-s)t*UV_D
其中：
- A,B,C,D 是梯形顶点
- UV_A,UV_B,UV_C,UV_D 是顶点纹理坐标
- (s,t) 是参数空间坐标
- P 是梯形内的点

牛顿迭代法

通过牛顿迭代法求解非线性方程：
1. 计算残差 R = P - F(s,t)
2. 计算雅可比矩阵 J = [dF/ds, dF/dt]
3. 更新参数: [s,t] = [s,t] - J⁻¹R
4. 迭代直至收敛
